## Image Optimization Tips

1. **Format**
   - Use WebP for best compression
   - Fallback to JPG for photos
   - Use PNG for logos/transparent images

2. **Size**
   - Player photos: Max 800x800px
   - Hero images: Max 1920x1080px
   - Logos: 200x200px to 400x400px

3. **Compression**
   - Use tools like TinyPNG or ImageOptim
   - Target: <200KB for photos, <50KB for logos

4. **Naming Convention**
   - Use lowercase
   - Use hyphens, not spaces
   - Be descriptive: `player-marcus-silva.jpg`
